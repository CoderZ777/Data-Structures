What is a stack?

A stack is a data structure similar to a queue but follows a different rule. It follows the rule, LIFO. LIFO stands for Last In First Out. Think of it as a stack of papers or plates; The most recent item you add to the stack, the most recently it will become removed. Adding an item to a stack is called pushing, while removing the top item is called popping.

1. Function Push(Argument ToInsert)
2. Check if the stack is full.
3. If the Front is equal to 0 and the Rear is equal N-1, the stack overflows and returns.
4. Otherwise, set the Rear to Rear + 1.
5. Set the Stack's Front to the ToInsert.
6. Return the function.


7. Function Pop
8. If the stack is empty, the stack overflows and returns.
9. If there is currently only one item, delete that item and note that the stack is empty.
10. Otherwise, delete the first item in the stack and return.
