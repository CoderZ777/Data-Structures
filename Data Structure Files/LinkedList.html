What is a Linked List?

A linked list is a collection of items called Nodes. Each node stores a value as well as another node called 'next'. This is the next item in the list. Chaining all of these Nodes together, all of these items form a rudimentary list.

Psudocode for Linked List(Click me!)

1.Class Linked List
2.Store a Node called Head
3.Store a Node called Tail


4.Class Node
5.Store an integer called Value
6.Store a Node called Next


7.Function AddItem(arguments: A linked list and a value to add)
8.If the linked list has nothing in it, add a head Node storing value and pointing to null
9.Otherwise, make the current tail point to a new Node storing the value
10.That node becomes the tail


11.Function AddItemAfter(arguments: A linked list called List, a value to add called Value, a node that should be before the adding node called After)
12.If the linked list has nothing in it, add a head Node storing value and pointing to null
13.And if the tail is the value to add that item after, run function AddItem with Value and stop this function. 14.Otherwise, make a new node storing Value and After's next value
15.Make after's next value the new Node we made storing value.
