Coding Terms!

Please note and remember these coding terms. If it helps, write them down. YOU MUST REMEMBER THEM!!!

  • Agile Software Development: A process where developers build software in stages.
  • Server: A host of websites and data.
  • Byte: 8 bits.
  • Bit: The smallest unit of information in a computer(1 or 0)
  • Inheritance: The practice of creating a new piece of code based on an older piece.
