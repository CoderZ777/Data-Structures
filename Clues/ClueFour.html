Basics of C++

C++ is a pretty cool programming language. It is one of the most used programs to create robots! Now, isn't that cool? However, it is what we call a "low-level" language. That basically means that it is easy for a computer to digest. High-level programming languages such as python need many different "decoding" steps to break down into something that a computer can read, c++ doesn't need as many of these steps.
